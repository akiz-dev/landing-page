## 🔹 Descrição

Este projeto é uma **landing page moderna** criada com **HTML5** e **CSS3** puros, inspirada em layouts corporativos. Desenvolvido como exercício de prática de front-end no **Odin Project**, o projeto inclui:

- Estrutura semântica clara (`header`, `main`, `footer`)  
- Layout responsivo com `flexbox`  
- Tipografia moderna via **Google Fonts**  
- Botões interativos com efeitos de hover  
- Seções clássicas de landing pages: Hero, Cards, Testemunho, Call-to-Action, Footer
